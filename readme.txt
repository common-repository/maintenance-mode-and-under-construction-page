== Description ==

Maintenance Mode & Under Construction Page which will manage your upcoming Launching of website & site offline page. Maintenance Mode plugin is provide you creative and well-crafted Construction page & Maintenance Mode Page. If your site is under development then use plugin to display an under maintenance / under construction page to your visitors. The plugin allows collecting all visitors subscriptions. When your site will live plugin notify all your active subscriber about site launch.

The ultimate Under Construction & Maintenance Mode plugin for WordPress websites which are coming soon. Maintenance mode under construction landing page is an outstanding WordPress plugin used by users when their website is under maintenance or is going under development. Create a simple page for Under Construction website or a maintenance mode page. Work on your site in private while visitors see a "Coming Soon" or "Maintenance Mode" page. Its a simple, no-nonsense plugin for all those situations when you have to hide the site behind an Under Construction page for a while.

== Installation ==

1. Upload the entire `maintenance-mode-and-under-construction-page` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Weblizar Coming Soon menu page to customize plugin settings.
4. Live it & Enjoy.
